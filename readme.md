# Dotfiles
my dotfiles

1. install [GNU stow](https://www.gnu.org/software/stow/) and [oh-my-zsh](https://github.com/ohmyzsh/ohmyzsh)
2. clone this repository into ~/dotfiles and do
```sh
  cd ~/dotfiles
```
3. run `make all` or `stow --verbose --target=$HOME --restow */`
